$emit(イベントのみ)
親コンポーネント 
<button @change="show=true">BUTTON</div>

data() {
    return {
        show: false
    }
}
子コンポーネント 
<button @click="$emit("change")">BUTTON</div>
$emit(イベントと値)
親コンポーネント 
<BaseButton @change="eventMethodChange">BUTTON</BaseButton>

data() {
    return {
        key: null
    }
}
methods: {
    eventMethodChange(param) {
        this.key = param
    }
}
子コンポーネント 
<button @click="eventMethods"><slot /></button>

methods: {
    eventMethods() {
        this.$emit("change",this.$vnode.key)
    }
}

イベント
focus
focusはフォーカスされたときイベントが発火します
blur
blurはフォーカスが外れたときにイベントが発火
change
フォーカスされてから外れるまでの間に値が変化している場合にイベントが発火します。
mouseover
mouseoverはマウスカーソルを当てたとき mouseover, mouseoutは、子要素であるpタグやdivタグ、さらにその子要素のpタグにカーソルを当てたり外した場合でも親側のイベントが発火
mouseout
mouseoutはマウスカーソルが外れたときにイ mouseover, mouseoutは、子要素であるpタグやdivタグ、さらにその子要素のpタグにカーソルを当てたり外した場合でも親側のイベントが発火
mouseenter
マウスカーソルを当てたとき 自身の要素のみイベント
mouseleave
マウスカーソルが外れたとき 自身の要素のみイベント
mouseup
mousedownはマウスのボタンを押したとき
mousedown
mouseupはマウスのボタンを離したときに発
scroll
スクロール時に発火するイベントです。
submit
submiされたときに発火するイベントです
drag
対象要素をドラッグ可能（draggable）にした場合に、ドラッグ開始（dragstart）、ドラッグ中（drag）、ドラッグ終了（dragend）に該当するイベントになります。
dragstart
対象要素をドラッグ可能（draggable）にした場合に、ドラッグ開始（dragstart）、ドラッグ中（drag）、ドラッグ終了（dragend）に該当するイベントになります。
dragend
対象要素をドラッグ可能（draggable）にした場合に、ドラッグ開始（dragstart）、ドラッグ中（drag）、ドラッグ終了（dragend）に該当するイベントになります。
dragenter
こちらはドラッグされる側のイベントになります。ドラッグ中にカーソルが当たった時（dragenter）と外れたとき（dragleave）のイベントになります。
dragleave
こちらはドラッグされる側のイベントになります。ドラッグ中にカーソルが当たった時（dragenter）と外れたとき（dragleave）のイベントになります。
dragover
こちらもドラッグされる側のイベントになります。dragoverは対象要素の中でドラッグしている間発火され続けます。dropは対象要素の中でドラッグをやめたときに発火
drop
こちらもドラッグされる側のイベントになります。dragoverは対象要素の中でドラッグしている間発火され続けます。dropは対象要素の中でドラッグをやめたときに発火
